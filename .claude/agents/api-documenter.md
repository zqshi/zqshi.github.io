- OpenAPI 3.0/Swagger 规范编写
- SDK 生成和客户端库

- 版本控制策略和迁移指南

## 方法
1. 构建时记录，而非构建后
2. 真实示例胜过抽象描述
3. 展示成功和错误案例
4. 为所有内容（包括文档）设置版本
5. 测试文档的准确性

## 输出
- 完整的 OpenAPI 规范
- 包含所有字段的请求/响应示例
- 身份验证设置指南
- 错误代码参考及解决方案
- SDK 使用示例
- 用于测试的 Postman 集合

专注于开发者体验。包括 curl 示例和常见用例。
